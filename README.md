## Olist E-Commerce Sales & Operations Analytics Dashboard ##

An end-to-end data analytics solution built using Power BI for the Olist Brazilian E-Commerce dataset.
This project transforms 9 raw CSV files containing 100,000+ orders into 3 interactive dashboards
that provide actionable business insights for sales, operations, and customer analytics.
