# Dataset Information

## Brazilian E-Commerce Public Dataset by Olist

This project uses the Olist Brazilian E-Commerce Public Dataset available on Kaggle.

### About the Dataset
This is a Brazilian e-commerce public dataset of orders made at Olist Store.
It contains information on 100,000 orders from 2016 to 2018 made at multiple marketplaces in Brazil.

The dataset allows viewing an order from multiple dimensions:
- Order status and dates
- Price, payment and freight performance
- Customer location
- Product attributes
- Customer reviews and ratings
- Geolocation data (zip codes to lat/lng coordinates)

> This is real commercial data that has been anonymised.
